# Práctica de MLOps_LLMOps - Guillermo Barrio #

La práctica consiste en una decena de ficheros que se dividen entre los apartados del enunciado.

**MLFlow**

- **Practica_MLOps_LLMOps_MLFlow.ipynb** - Notebook donde desarrollamos MLFlow a partir del dataset de aiabnb que usamos en la práctica de Machine Learning.

- **Capturas_Pantallas_MLFlow.pdf** - Capturas de pantallas de MLFlow

- **airbnb_test.csv** y **airbnb_train.csv** - Datasets necesarios para la práctica

**Scripts**

- **Scripts.py** - Fichero que incluye las funciones que definimos a partir del notebook de MLFlow, y que utilizamos en el de FastAPI

- **Scripts_NB.ipynb** - Cuaderno de apoyo para hacer el fichero Scripts.py

**FastAPI**

- **Practica_MLOps_LLMOps_FastAPI.py** - Fichero main de la aplicación

- **Capturas_Pantallas_MLFlow.pdf** - Captura de pantallas de FastAPI

- **Llamadas_curl_python.ipynb** - Notebook con las llamadas de http

**Despliegue de la App**

- **Capturas_Pantallas_Aplicacion_GCP.pdf** - Captura de pantallas del despliegue de la App

La web de la app es: https://airbnb-api-b2nrnqkdvq-ez.a.run.app/docs
